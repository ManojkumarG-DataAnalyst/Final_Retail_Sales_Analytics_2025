# Final_Retail_Sales_Analytics_2025
Final End-to-End retail sales analytics in Python (2M+ rows, RFM, retention, ABC, churn).


Retail and E-Commerce Data Analysis Project

Introduction:

This project is an end-to-end data analysis case study on a retail and e-commerce dataset using Python.

The objective of this project is to analyze customer behavior, product performance, and revenue patterns by applying practical business analytics methods commonly used in the retail domain.
